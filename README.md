
# SinergIA

## Descripción
En nuestra plataforma, mejorar tu comunicación nunca fue tan fácil. Con la ayuda de inteligencia artificial, podcasts, simulaciones con expertos y servicios personalizados, te ofrecemos todo lo que necesitas para avanzar en tus relaciones personales, estudios y carrera profesional. Ya sea que quieras dominar una entrevista laboral, mejorar tus conversaciones familiares o desarrollar tus ideas, aquí encontrarás la guía perfecta para lograrlo. ¡Es tu momento de crecer!

## Funcionalidades

### Conversaciones: Mejora tus relaciones
¿Estás preparado para tu próxima conversación importante? Ya sea con tu pareja, familia o amigos, practicar puede hacer la diferencia. Nuestra IA y simulaciones guiadas por expertos te preparan para esos momentos cruciales, ayudándote a comunicarte mejor, expresar tus ideas y fortalecer tus relaciones. ¡Deja de adivinar qué decir y asegúrate de que tus palabras impacten de verdad! Empieza a practicar hoy mismo.

### Impulsa tu vida: Convierte ideas en proyectos reales
¿Tienes una gran idea pero no sabes por dónde empezar? En Impulsa tu vida, nuestra IA actúa como mentor, mientras que especialistas y podcasts inspiradores te ofrecen las herramientas necesarias para transformar tus ideas en proyectos reales. Desde el primer paso hasta el último detalle, tendrás la orientación que necesitas. No dejes que tus ideas se queden en tu mente: ¡hazlas realidad ahora!

### Ayuda escolar: Aprende como nunca
¿Necesitas apoyo en tus estudios? No solo te ayudamos a entender tus materias, sino que te ofrecemos recursos interactivos y podcasts con explicaciones claras para cada nivel. Nuestra IA te guía paso a paso en Matemáticas, Lengua, Sociales y Naturales. Además, con simulaciones personalizadas, estudiar nunca ha sido tan sencillo y efectivo. Conviértete en tu mejor versión como estudiante. ¡Empieza hoy!

### Entrevista laboral: Sobresale en tus entrevistas
¿Nervioso por tu próxima entrevista? Aquí no solo te preparamos para responder preguntas, sino que te ayudamos a sobresalir. Practica con nuestra IA, recibe tips exclusivos de expertos y simula entrevistas reales para llegar con confianza y seguridad. También puedes acceder a podcasts donde profesionales comparten sus mejores consejos. No dejes tu futuro al azar: prepárate y destaca. ¡Hazlo ahora!

## Instalación
1. Asegúrate de tener [Node.js](https://nodejs.org/) instalado en tu sistema (versión 22.0).
2. Asegúrate de tener [PostgreSQL](https://www.postgresql.org/) como base de datos.
3. Clona este repositorio y accede a la carpeta del proyecto.
4. Ejecuta el siguiente comando para instalar las dependencias:

    ```
    npm install
    ```

5. Para iniciar el servidor en modo de desarrollo, ejecuta:

    ```
    npm run dev
    ```

6. O para iniciar la aplicación directamente:

    ```
    node index.js
    ```

## Requisitos previos
- Node.js v22.0 o superior
- npm v10.5.1 o superior
- PostgreSQL
